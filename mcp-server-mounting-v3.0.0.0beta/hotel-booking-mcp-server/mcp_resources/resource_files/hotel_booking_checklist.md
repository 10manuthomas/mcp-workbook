# Hotel Booking Checklist

## 1. Core Logistics
- [ ] **Location:** Is it near my primary destination or transport?
- [ ] **Price:** Does the final total include all taxes and resort fees?
- [ ] **Dates:** Are check-in and check-out times compatible with my travel schedule?

## 2. Policy & Security
- [ ] **Cancellation:** What is the deadline for a full refund?
- [ ] **Security:** Does the hotel have 24/7 staff or secure access?
- [ ] **Reviews:** Have there been recent complaints about cleanliness or safety?

## 3. Room & Comfort
- [ ] **Bedding:** Is the bed configuration correct for my group?
- [ ] **WiFi:** Is high-speed internet included and free?
- [ ] **Breakfast:** Is it included, or is there a nearby alternative?

## 4. Special Needs
- [ ] **Accessibility:** Are there elevators or ground-floor rooms if needed?
- [ ] **Parking:** Is on-site parking available and what is the cost?
